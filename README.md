# getNTLMInfoFromNTLMAuthenticationChallenge

NTLM information disclosure and OS fingerprinting tool.

Supports:
- HTTP / HTTPS
- SMB
- RDP (NLA)
- SMTP

## Installation

```bash
pipx install .
``
